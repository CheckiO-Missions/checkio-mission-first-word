Ти можеш почати з видалення всіх крапок і ком, потім ти зможеш отримати список усіх слів і знайти серед них перше.
Не міг би ти розбити це пояснення на менші кроки?
Перш за все, ти можеш використовувати str.replace() метод для заміни крапок і ком пробілами.
Чи можеш ти показати мені, як ним користуватися?
"replace" працює так: 
text = text.replace('.', ' ').replace(',', ' ')
Наприклад:
text = ".,.hello dude.,."
потім ми використовуємо наведений вище код і отримуємо:
" hello dude "
Що далі робити?...
Коли у тебе є чистий рядок лише зі словами та пробілами між ними, використай цей метод .split().
Я на півдорозі. Потрібна допомога!
Просто введи: 
text = text.replace('.', ' ').replace(',', ' ')
text = text.split()
І ти отримаєш список слів - ["hello", "dude"]
Мені потрібна маленька підказка.
Останнє, що ти повинен зробити, це знайти перший елемент списку.
І остаточна відповідь...
Повний код: 
def first_word(text):
    text = text.replace('.', ' ').replace(',', ' ')
    text = text.split()
    return text[0]
