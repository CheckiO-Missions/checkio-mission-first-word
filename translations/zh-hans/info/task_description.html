我可能已经看到了这个任务的简化版本First Word (simplified) 。这个任务有点复杂，因为在给定的字符串中不仅可以有英文字母。

你将得到一个字符串，你必须找到它的第一个单词。

解决任务时要注意以下几点：

  • 一个字符串中可以有点和逗号。
  • 一个字符串可以以一个字母开始，也可以是一个/多个点或空格。
  • 一个词可以包含一个撇号，它是一个词的一部分。
  • 整个文本可以用一个词来表示，就这样。

输入：一个字符串 (str{% if is_js %}ing{% endif %}).

输出：一个字符串 (str{% if is_js %}ing{% endif %}).

例子： 

{{init_code_tmpl}}

它是如何使用的： 第一个词是命令行中的一个命令

前提条件： 该文本可以包含a-z A-Z , .
