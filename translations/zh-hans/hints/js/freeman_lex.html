💡 这个想法包括以下步骤：将所有的点和逗号换成空位，修剪所有的开头和结尾的空白处，用剩余的空白处（字与字之间）分割文本，取第一个字（别忘了JS中的0索引）。

好吧，我应该从什么开始呢？

阅读关于replace() 方法。用它来替换所有的点和逗号（下一个提示），用一个空的空间。 

text.replace(..., " ")

那么，如何找到文本中的所有点和逗号来替换它们？

你需要一个regular expression 来做到这一点。建立以下正则表达式(/.../)：带转义字符的点号\. ( 因为点号在正则表达式中意味着任何单个字符，你需要转义它来寻找点号本身)或| 逗号, 。你的正则表达式应该有全局修饰符/.../g ，以查找文本中的所有点和逗号。 

text.replace(/\.|,/g, " ")

好了，接下来该怎么做？

使用trim() 方法，在替换后去除文本开头和结尾的所有空白处。 

text.replace(/\.|,/g, " ").trim()

如何将文本分割成不同的字？

使用split() 方法，在修剪后使用空白" " 参数。 

text.replace(/\.|,/g, " ").trim().split(" ")

我现在得到的是什么，如何取第一个字？

分割后，你得到一个独立的单词数组。因此，为了得到第一个词（它的索引是0 ），你需要在之前的所有步骤之后使用索引。 

text.replace(/\.|,/g, " ").trim().split(" ")[0]

那么，我的最终解决方案应该是什么样子的呢？

你只需要把你的代码包装成函数表达式，并返回上面的整个表达式，因为你没有声明任何变量。

下面是整个解决方案：

function firstWord(text: string): string {

    return text.replace(/\.|,/g, " ").trim().split(" ")[0]
}
