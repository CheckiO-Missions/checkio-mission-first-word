Прежде всего, попробуйте использовать метод str.replace() чтобы заменить точки и запятые пробелами.
Вы могли бы показать, как это используется?
В данном случае код может выглядеть так: 
        text = text.replace('.', ' ')
        text = text.replace(',', ' ')
Например, если наш исходный текст:
text = ".,.hello dude.,."
мы используем код, написанный выше и получаем:
" hello dude "
Замена именно на пробелы, а не на какие-нибудь другие символы, очень упростит работу со строкой в дальнейшем, когда её нужно будет разбить на отдельные слова. Также, этот код может быть записан в одну строку и работать аналогично: 
        text = text.replace('.', ' ').replace(',', ' ')
Что дальше?
Теперь, когда у вас есть строка, состоящая только из слов, разделенных пробелами, вы можете использовать метод split. Он преобразует строку в list, убрав все пробелы и оставив только слова. При этом каждое слово будет отдельным элементом списка и вам нужно будет лишь найти первый элемент (первое слово).

Полная версия кода выглядит так: 
        def first_word(text):
            text = text.replace('.', ' ').replace(',', ' ')
            text = text.split()
            return text[0]
и работает следующим образом: на входе ваша функция получает строку, затем заменяет все точки и запятые пробелами, после этого метод split() удаляет все пробелы и преобразует вашу строку в list, где каждый элемент списка - это отдельное слово исходного текста. И в конце ваша программа возвращает первый элемент списка, то есть - первое слово исходного текста.
