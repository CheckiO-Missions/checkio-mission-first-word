üí° The idea includes the following steps: to replace all dots and commas with empty space, trim all beginning and end whitespace, split the text by remaining whitespace (between words) and take the first word (don't forget about 0-indexing in JS).

Ok, what should I start from?

Read about replace() method. Use it to replace all dots and commas (next hint) with an empty space. 

text.replace(..., " ")

So, how to find all dots and commas in text to replace them?

You need a regular expression to do so. Build the following regular expression (/.../): dot sign with escape character \. (since dot means any single character in regex you need to escape it to look for dot as character itself) or | comma sign ,. Your regular expression should have global modifier /.../g to find all dots and commas in the text. 

text.replace(/\.|,/g, " ")

Ok, what to do next?

Use trim() method to remove all whitespace form the beginning and end of the text after replacing. 

text.replace(/\.|,/g, " ").trim()

How to split the text to the separate words?

Use split() method with whitespace " " argument after trimming. 

text.replace(/\.|,/g, " ").trim().split(" ")

What I get for now and how to take the first word?

After splitting you get an array of separate words. So, to get the first one (it has index 0) you need to use indexing after all previous steps. 

text.replace(/\.|,/g, " ").trim().split(" ")[0]

So, how my final solution should look like?

You just need to wrap your code into function expression and return the whole expression above, since you don't declare any variables.

Here is the whole solution:

function firstWord(text: string): string {

    return text.replace(/\.|,/g, " ").trim().split(" ")[0]
}
